> — Jorge López (@jlopezper) [August 29, 2019](https://twitter.com/jlopezper/status/1167040581236854785?ref_src=twsrc%5Etfw)

Por medio de una conversación en Twitter he llegado al paquete [opendataes](https://github.com/rOpenSpain/opendataes) para acceder mediante R a la API de datos.gob.es Este paquete desarrollado inicialmente por Jorge López **necesita colaboraciones** porque tenemos a nuestro alcance más de 22.000 conjuntos de datos de acceso libre para realizar nuestros propios estudios.

Conocía [rOpenSpain](https://github.com/rOpenSpain/) por [MicroDatoEs](https://github.com/rOpenSpain/MicroDatosEs) y tengo pendiente trabajar con [CatastRo](https://github.com/rOpenSpain/CatastRo) pero ya estáis tardando en ejecutar remotes::install_github(«ropenspain/opendataes») e instalar opendata, además tenemos que colaborar para mejorar las consultas a la API porque se abre un mundo de datos accesible desde R brutal.
