Muchos hemos sufrido este error trabajando con Enterprise Guide de SAS:

Unable to open file as a valid project file

Además de la pantalla:

![eguide-version-2.PNG](/images/2010/09/eguide-version-2.PNG)

[De este modo «tan sencillo» ](http://www.dnmca.com/forum/index.php?topic=183.0)podemos recuperar los códigos de nuestro proyecto de Enterprise Guide siempre que sean proyectos *.egp y no *.seg. Por cierto, para abrir proyectos de versiones anteriores tenemos el Migration Wizard de Guide, da algunas pegas prefiero este método. Un gran truco de [http://www.dnmca.com](http://www.dnmca.com/)
