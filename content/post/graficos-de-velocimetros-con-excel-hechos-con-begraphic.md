[En el blog ya hemos hablado de `Begraphic`](https://analisisydecision.es/begraphic-un-add-in-para-excel-muy-interesante/). Ahora vamos a descubrir algunas de sus posibilidades. En concreto vamos a realizar gráficos del tipo velocímetro con Excel de una forma muy sencilla. Tan sencilla que vamos a partir de tener instalado `Begraphic` y de estos datos:

![Velocímetro Excel 1](/images/2012/02/velocimetro_excel1.png)

En la pestaña `Begraphic` tenemos el complemento `Gauge` y empezaremos con el primer paso que nos sirve para ubicar el gráfico resultante:

![Velocímetro Excel 2](/images/2012/02/velocimetro_excel2.png)

Seleccionamos donde queremos que se genere el gráfico. Y el siguiente paso nos pregunta si deseamos mantener el tamaño, no he jugado con este paso:

![Velocímetro Excel 3](/images/2012/02/velocimetro_excel3.png)

Tras validarlo lo siguiente que nos encontramos es la ventana más importante:

![Velocímetro Excel 4](/images/2012/02/velocimetro_excel4.png)

En esta ventana es recomendable tener activado `Enable advanced mode` para descubrir todas las posibilidades que nos ofrece `Begraphic` con este tipo de gráficos. No es necesario que os diga que datos tenemos que introducir en `minimum`, `maximum` y `current value`. Ahora podemos seleccionar el tipo de gráfico, particularmente prefiero los semicírculos, para ello tenemos `Shape` :

![Velocímetro Excel 5](/images/2012/02/velocimetro_excel5.png)

A partir de aquí podemos jugar con todas las opciones que tiene y crear de forma muy sencilla gráficos espectaculares como este:

![Velocímetro Excel 6](/images/2012/02/velocimetro_excel6.png)

Y si no nos gusta su aspecto final sólo tenemos que situarnos sobre el gráfico y pulsar el complemento `Set parameters` y volveremos a la pantalla anterior donde retocaremos aquellos parámetros que deseemos. Sólo tenéis que instalar `Begraphic` y veréis lo sencillo que es la realización de este tipo de gráficos con este `add in`. En las próximas semanas veremos como hacer mapas con `Begraphic`. Espero poder hacer distintos mapas. Saludos.
